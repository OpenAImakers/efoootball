# ⚽ Welcome to [eFootball](https://efootballkenyaleague.website/)
> A competitive eFootball platform for creating leagues, tracking matches, and building your gaming career in a structured ecosystem.

## 🎯 Purpose
eFootball is built for mobile competitive efootball ecosystem players bringing structure, competition, and community into one platform.

## ✨ Features
- Tournament creation  
- Match voting and live updates  
- Leaderboard standings  
- User profiles and engagement  
- Follow users and tournaments  
- League management

## 🛠 Tech Stack
Built using:
- **React (Create React App)**
- **TypeScript**
- **JavaScript**
- **MySQL**

## ⚡ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/OpenAImakers/efoootball.git
  ```


2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npm start
   ```
You can now view e-football in the browser.

Local: http://localhost:3000
<p>
On Your Network:  http://172.20.121.116:3000
</p>

You can start developing by editing the files inside the **src** directory .
